# CSS Development Specs

This directory contains coding standards, guidelines, and best practices for CSS development within the ASDM ecosystem.

## Overview

CSS specs provide comprehensive guidance for:
- Writing clean, maintainable CSS code
- Using CSS preprocessors (Sass, Less) effectively
- Optimizing CSS for performance
