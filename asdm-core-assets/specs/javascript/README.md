# JavaScript Development Specs

This directory contains coding standards, guidelines, and best practices for JavaScript development within the ASDM ecosystem.

## Overview

JavaScript specs provide comprehensive guidance for:
- Writing clean, maintainable JavaScript code (ES6+)
- Following best practices for language features
- Effective commenting and documentation

## Available Specifications

### Coding Standards
**File**: [javascript.md](javascript.md)

Covers:
- Code style and formatting (2 spaces indentation)
- Semicolon usage
- Variable declarations
- Function best practices
- Object and array patterns
- ES6+ features
- Error handling
- Comments and naming conventions
